# chrome-tab-multiselect-extension
This extension adds shortcuts to multiselect tabs.

#### Shortcuts
chrome://extensions/shortcuts
* Multiselect Left: 
  * **Alt ⇧ ←** 
  * `MacOS` **⇧ ⌘ ←**
* Multiselect Right: 
  * **Alt ⇧ →** 
  * `MacOS` **⇧ ⌘ →**
* Multiselect Tab Group:
  *  **Alt ⇧ ↑** 
  * `MacOS` **⇧ ⌘ ↑**
